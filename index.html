Algunos lenguajes de la web

HTML + CSS + Javascript

  • HTML5

HTML5 (HyperText Markup languaje, versión 5) es la quinta versión importante del lenguaje básico de la Word Wide Web, HTML.

Usa la extesión .html
  • CSS

Hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets) es un lenguaje usado para definir y craer la presentación de un documento estructurado escrito en HTML.

Usa la extensión .css
  • JS

JavaScript (abreviado comúnmente ¨JS¨) es un lenguaje de programación interpretado utilizando principalmente en el lado del cliente (client-side), implementando como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas.

Usa la extensión .js
